## Simple To - Do List

This is a simple version of to do list i made (ik it is beginner level) i made it so that i understand how flask works

To run this at first install python libraries required
```pip install -r requirements.txt```

**or just**

```pip install flask```

Either way is fine in this case

and the just run
```python server.py```

after executing this go to the ip address mentioned in it and yeah u can just see the to - do list page
